    Services

    I offer various services designed to cater to the needs of businesses or individuals in need of websites.

    • Web Design and Development

      The most fundamental service is basic web design and development, ideal for brochure sites or portfolio-based websites, crucial for any business to showcase itself effectively.

    • UX & UI

      Linked closely with web development, user interface and user experience design involve creating a website that follows proper design practices in order to create an enjoyable and intuitive experience for the user. This includes aspects such as visual design, responsive design and typography.

    • Web Applications

      The web application is a more advanced type of website which offers the user a more interactive and "app-like" experience. Web applications generally rely on the retrieval of information from a database. This allows for a more customizable and powerful experience.

    • Progressive Web Apps

      Progessive web apps essentially websites configured to be offer some native application functionality such as offline functionality and push notifications. Progressive web apps are can be 'installed', similarly to traditional applications on Android phones. If you want to test this out, and you're viewing this page on an Android phone using Chrome or Firefox, tap the browser's menu button and select "Add to home screen".

    • Content Management Systems

      Content management systems provide a user-friendly interface that allows users, owners and administrators to easily create displayable content for a website. Most blogs use content management systems, as content creators can manage their content without any web development knowledge.

    • Data Visualisation

      Data visualisation allows data to be displayed in unique and customizable ways, making data more significantly more interesting than plain text. For examples of interesting data visualisations, visit the D3 gallery

      .

    • Logo Design

      Eye-catching and expressive logos are important for the establishment of a business's brand image. Following visual design and logo design guidelines, I provide logo designs with the purpose of creating a memorable brand image.

    • Search Engine Optimisation (SEO)

      Search engine optimisation is the process of optimising a website through multiple adjustments for the purpose of ranking a website higher in search engine results. This generates more traffic from users, as the website is made significantly more available to the public.
